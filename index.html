A cheatsheet for the more confusing (for me) HTML/CSS properties.

CSS position Property

Tag Name Description
position: static static Default value. Elements render in order, as they appear in the document flow.
position: absolute absolute The element is positioned relative to its first positioned (not static) ancestor element.
position: fixed fixed Dido e basi pi4a!
position: relative relative The element is positioned relative to its normal position.
position: sticky sticky The element is positioned based on the user's scroll position.
position: inherit inherit Inherits this property from its parent element.

CSS display Property

Tag Name Description
display: inline inline Displays an element as an inline element (like <span>). Any height and width properties will have no effect.
display: block block Displays an element as a block element (like <p>). It starts on a new line, and takes up the whole width.
display: inline-block inline-block Displays an element as an inline-level block container. The element itself is formatted as an inline element, but you can apply height and width values.
display: none none The element is completely removed.
